# String Patterns

##  1. Two Pointers + Basic String Handling


**Core idea:** Left-right movement, in-place logic

**Revise these:**

* Valid Palindrome
* Reverse Words in a String
* Remove Consecutive Characters
* One-removal palindrome

**What you must be able to do:**

* Handle edge cases (spaces, symbols, casing)
* Modify string *without extra space* when needed

##  2. Hashing + Frequency Patterns

**Core idea:** Count → Compare → Group

**Revise:**

* Valid Anagram
* Print duplicates
* Group Anagrams
* Sort by frequency

**Key instinct:**

* “Can I convert this into a frequency map problem?”

##  3. Sliding Window (MOST IMPORTANT)

This is your **interview goldmine**.

**Revise deeply:**

* Longest substring without repeating characters
* Longest repeating replacement
* Minimum window substring
* Count anagrams in substring
* At most K distinct

**What you must master:**

* Expand → violate → shrink → fix
* When to use `while` vs `if`
* Map vs set usage
